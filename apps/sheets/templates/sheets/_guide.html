Setting up a Google Sheets connection

Google Sheets connections are simple to set up, the only thing that needs to happen is to share the sheets URL and give us access to it.

We'll guide you through the steps to get your sheets connected.

Step 1

Open the Google Sheets file that you intend to connect and copy the URL into the URL input field on this page.

Step 2

Now we need to sort out the access to the file. We accomplish this by sharing the sheet file with the following email:

{{ service_account }}

To give access to this account first locate the Share button on the Google Sheets page and click it.

In the modal that opens paste the email and make sure to give it Viewer access.

Once that is set up hit the Send button. If another window pops up to confirm to share click Share anyway.

Optional: Step 3

To select a specific cell range that you want to import you can go to the Google Sheets page with your sheet and select the desired range. After selecting the desired range you can copy the value in the top left corner into the Cell range field on this page.

Note: If you want to select a cell range from a specific sheet make sure to add the sheet name in front of the cell range followed by a !, like so:

My Sheet!B2:C3

Step 4

Last part now is to create the integration. We will test access to the sheets to verify the connection works. If there is an issue retry the steps, in case you encounter problems please do reach out to us!
