Merge two tables side by side meaning the columns of the second table will be added to the first. Select the a column from each table that make up your join criteria. Columns are selected in the order of connected parent nodes. Then select the mode of joining.

  • Inner: Only columns for which the join condition is full-filled are kept.
  • Outer: All columns from both tables are kept. When the join condition isn't full-filled the columns are filled with nulls.
  • Left: All columns from the first (left) table are kept, columns from the right table are filled with nulls if join-condition is not full-filled
  • Right: All columns from the second (right) table are kept, columns from the left table are filled with nulls if join-condition is not full-filled
